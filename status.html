วิธีตั้งค่า Google Sheet (ครั้งแรก)
  1. สร้าง Google Sheet หัวคอลัมน์: application_id, name, course, paid, status, updated_at
  2. ไปที่ File → Share → Publish to the web เลือกชีตที่ต้องการ แล้วกด Publish
  3. คัดลอก Sheet ID (อยู่ใน URL หลัง /d/) และ gid ของชีตนั้น
  4. เปิดไฟล์ assets/app.js แล้วแทนค่า SHEET_ID และ GID
  5. Commit & Push ไปที่ GitHub Pages จากนั้นรีเฟรชหน้านี้
