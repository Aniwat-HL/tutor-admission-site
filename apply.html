สมัครเรียน

คำชี้แจงก่อนสมัคร (Before You Apply)

1) กรอกฟอร์มสมัคร → 2) เปิดฟอร์มอัปโหลดสลิปในแท็บใหม่ → 3) ใช้ Application ID เดียวกัน ทั้งสองฟอร์ม
Fill in the application form, then open the upload form in a new tab and use the same Application ID.

รหัสผู้สมัคร (Application ID)

โปรดคัดลอก Application ID นี้ไปกรอกในฟอร์ม “สมัครเรียน” และ “อัปโหลดสลิป” เพื่อให้เจ้าหน้าที่จับคู่ข้อมูลได้
Please copy this Application ID into both forms so we can match your records.

* ระบบจะสร้าง ID ให้อัตโนมัติและเก็บในเครื่องของคุณชั่วคราว (localStorage)

ขั้นตอนที่ 1: กรอกแบบฟอร์มสมัครเรียน

กรอกข้อมูลให้ครบถ้วน และอย่าลืมใส่ Application ID จากด้านซ้ายในช่องที่กำหนด
Fill in the form and include your Application ID in the designated field.

ขั้นตอนที่ 2: อัปโหลดสลิป/หลักฐานการชำระเงิน

  1. ตรวจสอบให้แน่ใจว่าได้กรอก “Application ID” ไว้ในขั้นตอนที่ 1 แล้ว
  2. กดปุ่มด้านล่างเพื่อเปิดฟอร์มอัปโหลด (จะเปิดในแท็บใหม่)
  3. กรอก Application ID เดียวกันแล้วแนบสลิป
อัปโหลดสลิป/หลักฐานการชำระเงิน (Open Upload Form)

* ฟอร์มอัปโหลดไฟล์ของ Google จะไม่สามารถฝังในเว็บได้ แต่สามารถเปิดในแท็บใหม่ได้ตามปกติ
